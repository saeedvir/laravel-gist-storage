# Changelog

All notable changes to `laravel-gist-storage` will be documented in this file.

## v1.0.0 - 2025-11-21

### Added
- Initial release
- Laravel 11/12 support
- Flysystem v3 adapter
- Full Storage facade integration
- Livewire file upload support
- PSR-compatible code
- GistClient with full API support
- Configuration publishing
- Comprehensive documentation
- PHPUnit test setup

### Features
- Store files on GitHub Gist
- Read/write/delete operations
- Stream support
- File metadata retrieval
- Copy and move operations
- List files in gist
- Multiple gist management via GistClient
